CSS Techniques Page

1. Blurry Text

The blurry text technique makes the color of the text transparent and adds a shadow.

More browsers spuport color than text-shadow though, so you might want to do feature detection. Or, leave the color property and do enough shadowing that it appears blurry anyway.
- Quote and technique from CSS Tricks
